# Executive-Memory Drift Capture

When session evidence shows drift or incompleteness in an executive-memory
surface, capture that observation in `.agent/memory/active/napkin.md` with the
tag `Source plane: executive`.

Executive memory is stable catalogue memory. It drifts quietly unless active
work routes fresh evidence back into it.

Use `consolidate-docs` to graduate repeated or settled executive-memory
observations into the affected executive surface.
