# Research

Default exploratory-synthesis tier.

Use this directory for:

- topic surveys
- draft comparisons
- evidence gathering that is not yet ready for `analysis/`
- material that may later be promoted into `analysis/`, `reports/`, or
  `reference/`

## Routing rule

- if the work is still exploratory, it belongs here
- if the evidence has stabilised into a focused investigation, move or promote
  it into `analysis/`
- if it becomes a formal audit or synthesis, promote it into `reports/`
- if it becomes evergreen library material, promote it into `reference/`

This template keeps the research tier intentionally light until real repo work
produces durable investigative material.
