# Reference

Curated library tier. Material here is deliberately promoted, evergreen, and
owner-vetted.

Governed by
[PDR-032](../practice-core/decision-records/PDR-032-reference-tier-as-curated-library.md).

## What lives here

A reference document is:

- deliberately promoted from another surface
- evergreen enough to stay useful across sessions
- owner-vetted as read-to-learn material

Fresh material does not start here by default. It starts in `research/`,
`analysis/`, `reports/`, or active memory and is promoted only when it clearly
meets the bar.

## Promotion rule

Before promoting a document into `reference/`, make three things explicit:

1. why this is reference and not research
2. why the substance is evergreen
3. why this location is the right library home
