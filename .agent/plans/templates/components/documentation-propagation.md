# Component: Documentation Propagation

No phase is complete until durable documentation has been handled.

## Required handling

Plans should explicitly record whether they changed:

1. `.agent/practice-core/practice.md`
2. `.agent/practice-index.md`
3. the relevant ADRs or repo docs
4. affected README files

If no update is needed, record a brief no-change rationale.

Before closure, apply `jc-consolidate-docs` or the equivalent canonical
consolidation workflow.
