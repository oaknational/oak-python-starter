# Reports

This directory holds promoted report artefacts: formal audits, stable
syntheses, and report-shaped outputs that should remain distinct from raw
research or working analysis.

## Authority split

- `.agent/research/` owns exploration and source gathering
- `.agent/analysis/` owns investigations and evidence
- `.agent/reports/` owns promoted reports once they are stable enough to stand
  alone

## Typical report shapes

- repo-health audits
- formal integration reports
- promotion-ready syntheses that consolidate multiple analyses

When a document is still gathering evidence or changing shape materially, it is
not a report yet.
