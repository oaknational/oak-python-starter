# Practice Core Patterns

This directory is reserved for portable general patterns that are abstracted
from concrete repo evidence.

## Scope

Put a pattern here only when it is:

1. ecosystem-agnostic
2. engineering-shaped rather than Practice-governance-shaped
3. supported by multiple concrete instances

Portable governance belongs in `../decision-records/`. Repo-local proven
instances belong in `.agent/memory/active/patterns/`.

## Current State

This directory starts intentionally light. The Python template is carrying the
surface and contract first; portable general patterns can be added once enough
cross-repo evidence has accumulated.
